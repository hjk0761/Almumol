## 💡 풀이 방법 요약

dp 문제 입니다.

각 프로그램 시청을 완료하는 시점에 만족도를 오른다고 생각하면, dp를 종료 시간에 따른 최대 만족도라고 설정할 수 있습니다.

프로그램이 끝나는 시간(시작시간 + 지속시간)을 기준으로 정렬한 후, 지금 시간에 프로그램을 시청하는것(시작시간까지의 dp 값 + 이 프로그램을 시정할 때의 만족도) 시청하지 않는 것(지금 시간의 dp 값) 중 큰 값을 가져가면 됩니다.
