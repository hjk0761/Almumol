## 💡 풀이 방법 요약

브루트포스를 사용하였습니다.

문제 제한은 n, m이 최대 8이므로 벽을 세울 세 좌표를 브루트포스를 통해 찾아도 64^3 = 262144 밖에 되지 않습니다.

따라서 벽을 세울 각 좌표를 설정한 후, BFS 를 통해 바이러스가 퍼지게 한 후 빈 칸의 개수를 세주면 됩니다.
